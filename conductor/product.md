# Product Guide

## Initial Concept
A collaborative, standalone financial tracking application designed for shared household expenses.

## Target Audience
Households, roommates, and small groups who need to track shared expenses, view balances, and settle debts easily.

## Core Features
- **Group Management:** Create and manage groups for different shared contexts (e.g., apartments, trips).
- **Expense Tracking:** Add shared expenses with support for various split methods (e.g., equally, by percentages, exact amounts).
- **Balance Calculation:** Automatically calculate individual balances and simplify debts to minimize the number of transactions needed to settle up.
- **Settlements:** Record payments to settle balances between members.
- **User Authentication:** Secure user profiles and group access.
